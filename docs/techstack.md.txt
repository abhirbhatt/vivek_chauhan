# TECH_STACK.md
High-Performance Tech Stack for Cinematic Filmmaker Portfolio

--------------------------------------------------
1. CORE STACK (FRONTEND)
--------------------------------------------------

Framework:
Next.js 14 (App Router)

Why:
- Server Components
- Built-in image & video optimization
- Code splitting
- Streaming rendering
- SEO friendly

Language:
TypeScript

Why:
- Type safety
- Fewer runtime bugs
- Better scaling

--------------------------------------------------
2. STYLING SYSTEM
--------------------------------------------------

Tailwind CSS

Why:
- Utility-first (fast development)
- Tree-shaken
- Small final CSS bundle
- Easy glassmorphism
- Responsive control

PostCSS + Autoprefixer

--------------------------------------------------
3. ANIMATION ENGINE
--------------------------------------------------

Primary:
GSAP (GreenSock)

Plugins:
- ScrollTrigger
- Flip
- Observer

Why:
- Industry standard for cinematic motion
- 60fps animations
- Timeline control
- Precise staggering
- Perfect for video reveals

Secondary:
Framer Motion (micro interactions)

Why:
- Simple hover / tap animations
- Layout transitions

--------------------------------------------------
4. SMOOTH SCROLLING
--------------------------------------------------

Lenis (by Studio Freight)

Why:
- Hardware accelerated
- Natural inertia scrolling
- Works with GSAP ScrollTrigger
- Mobile friendly

--------------------------------------------------
5. GLASS / BLUR EFFECTS
--------------------------------------------------

CSS:
backdrop-filter: blur(12px)

Tailwind utilities:
backdrop-blur-md  
bg-white/10  

Fallback:
solid color when not supported
