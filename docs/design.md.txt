# DESIGN.md  
Cinematic Filmmaker Portfolio – Design & Motion Specification  
(Reference-Based UI Replication Guide)

--------------------------------------------------
1. DESIGN INTENT
--------------------------------------------------

Create an immersive, cinematic, minimal portfolio that feels:

• Dark  
• Atmospheric  
• Motion-driven  
• Video-first  
• Editorial  
• Premium  

The website must feel like a film intro, not a normal webpage.

--------------------------------------------------
2. COLOR SYSTEM
--------------------------------------------------

--background-primary: #000000  
--background-secondary: #0A0A0A  

--text-primary: #FFFFFF  
--text-secondary: #CFCFCF  
--text-muted: #7A7A7A  

--overlay-dark: rgba(0,0,0,0.45)  
--border-subtle: rgba(255,255,255,0.12)

--------------------------------------------------
3. TYPOGRAPHY
--------------------------------------------------

Primary Font:
Inter / Helvetica Neue / Neue Haas Grotesk

Weights Used:
300, 400, 500, 600, 700

Scale:

Hero H1 → 80–110px  
Section H2 → 48–64px  
Card Title → 22–28px  
Body → 16–18px  
Small Label → 12–13px  

Letter Spacing:
H1: -0.03em  
H2: -0.02em  
Body: 0em

7. LOADER (PAGE INTRO)
--------------------------------------------------

8. CUSTOM CURSOR (OPTIONAL)
--------------------------------------------------

9. NAVIGATION BAR
--------------------------------------------------

Transparent initially

Logo left

fade + slide down

Underline grows left → right

10. HERO SECTION
--------------------------------------------------

H1  
Subheading

--------------------------------------------------
23. GLOBAL TRANSITIONS
--------------------------------------------------

All interactive elements:

--------------------------------------------------
24. SCROLL ANIMATION SYSTEM
--------------------------------------------------

Elements start with:

opacity:0  
transform: translateY(40px)

When in view:

opacity:1  
transform:none  

Use IntersectionObserver or GSAP ScrollTrigger

--------------------------------------------------
25. VIDEO RULES
--------------------------------------------------

All muted  
No controls  
Lazy loaded  
Poster images required  

--------------------------------------------------
26. RESPONSIVE BEHAVIOR
--------------------------------------------------

<768px:

H1 → 44–56px  
Grid → 1 column  
Nav becomes hamburger  

--------------------------------------------------
27. ACCESSIBILITY
--------------------------------------------------

Contrast AA+  
Keyboard focus visible  
prefers-reduced-motion respected  

--------------------------------------------------
28. PERFORMANCE RULES
--------------------------------------------------

Preview videos <5MB  
Hero video <15MB  
Images WebP  
Lazy loading

--------------------------------------------------
30. DESIGN GOAL
--------------------------------------------------

Every scroll feels like a film cut.  
Every hover feels alive.  
Nothing flashy.  
Nothing noisy.  
Only motion, typography, and imagery.
